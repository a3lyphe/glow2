Glow Speed Tests

Welcome to Glow's speed tests. Below you fill find a list of available tests to run. Each will show a comparison between the current unbuilt version of Glow 2.0 alpha against Glow 1.7.2, and in most cases against jQuery 1.4 or YUI 3.0, as best match methods apply.

To prevent extraordinary test times and browser freezes, each test is run over the period of 1 second, and the number of executions made by the given method in that time is recorded. We're using this number to benchmark best speed.

These pages should be tested on all available browsers and platforms, as results will vary.
